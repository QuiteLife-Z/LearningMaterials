# 人工智能

## 说明  
目前国内外主流的大模型如下：  
- 国内：通义千问、豆包等；  
- 国外：ChatGPT、Gemini、Grok 等。

本文以阿里巴巴的“通义千问”为例（这是我使用时间最长）。

## 优点  
1. 相较于豆包等其他国产大模型，文本输出更加自然、拟人化。  
2. 对于简单的代码问题，能够较为满意地解决。

## 缺点  
1. 在编写 HTML 代码时，虽然会提供图片链接，但无法保证图片内容完全一致或可用。  
2. 无法对视频内容进行总结。  
3. 信息存在滞后性（例如搜索研究生考试相关信息时，结果与官方发布内容差异较大）。  
4. 个人感觉很多内容未被完整呈现，仅选择性地输出部分信息。  
5. 有时 AI 模型会虚构方法、数据甚至引用不存在的资料。

## 主要问题  
实际收费与预期存在显著差异。

## 自主探索  
基于以上不足，我开始着手开发属于自己的人工智能系统（走上自我进化之路），并持续搜集学习资料、积累知识（计划每月更新进展）。

## 理想中的 AI 解决方案  
1. **一句话完成手机操作**：如点外卖、跨平台比价、高德地图导航路线规划、打车平台下单等。  
2. **AI 与行业深度融合**：例如集成计算机视觉、实现风险自动识别与检索。  
3. **办公自动化**：自动生成表格、Word 文档、PPT 演示文稿等制作。  
4. **智能学习辅助**：支持知识自主扩展与个性化学习路径构建。

目前，这个人工智能系统已启动自我进化流程，通过持续学习构建个性化的 AI 模型。
